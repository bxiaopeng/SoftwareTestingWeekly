# 软件测试周刊
展示一周内关于**软件测试**的一些资讯，内容开源，社区共建。

你可以推荐：

- 一些最新的动态
- 一段启发的言论
- 一种有效的方法
- 一篇极好的文章
- 一款开源的工具
- 一张内涵的图片

欢迎提交 [issue](https://github.com/bxiaopeng/SoftwareTestingWeekly/issues)。
